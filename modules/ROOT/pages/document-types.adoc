Configuring a message type for a partner means configuring settings specific to that document type standard.

== X12 and JSON

Refer to these when configuring either X12 or JSON

[%header,cols="3s,7a"]
|===
|Setting |Description

|Name

|Name for this document type

Required:: Yes

Default:: Auto-populates based on the following values:

* *Company name*
* *Standard*
* *Version*

|Standard

|Select a value from the drop-down menu.

Required:
Yes

Valid values::

`X12`
`JSON`

Default::

None

|Version - X12 only

|Select a value from the drop-down menu, or enter a different value.

Required:
Yes

|Message Types - X12 only

|Select a value from the dropdown listbox, or enter a different value.

Required:
Yes


|Schema

|Clicking the *Use Standard* button accepts the standard you selected, or clicking the *Custom* button allows you to choose a file to download. You must use a custom schema for JSON.

Required:
For JSON


|===
