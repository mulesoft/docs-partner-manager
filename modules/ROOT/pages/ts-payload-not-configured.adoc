= You cannot deploy this message flow because the payload storage is not properly configured

ifndef::env-site,env-github[]
include::_attributes.adoc[]
endif::[]


== Cause

Payload Storage is not configured.

== Solutions

Verify that your Payload Storage is active.

* From within your Anypoint Partner Manager, select your <Hostname> profile.
* Scroll down to the `Payload storage section` and click *Edit*.
* Review your payload storage configuration.
* If not configured, refer to https://docs.mulesoft.com/partner-manager/2.0/setup#setup-the-payload-storage-api[Set up the Payload Storage API].
